A simple timestamp microservice built with Spring Boot. A deployed version can be viewed
[here](https://slicetimestamp.herokuapp.com/).

## Running Locally
```
gradle bootRun
```
Then point your browser to [localhost:8080](http://localhost:8080).

## Running Tests
```
gradle test
```
